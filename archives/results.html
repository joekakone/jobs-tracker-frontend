Stagiaire Data Analyst

Yas Togo • Lomé, Togo

Description du poste

Le groupe TOGOCOM, dans le cadre du renforcement de son équipe analytics et BI, recherche pour un contrat à durée indéterminée un.e Stagiaire Data Analyst.

Missions principales :

  • Gérer des rapports d'analyse de données pour répondre aux besoins précis.
  • Étudier les données statistiques et produire des recommandations métiers.
  • Construire, interpréter et diffuser les rapports BI & Analytics.
  • Maîtriser les outils d'analyse pour suivre les performances de l'activité.

Compétences requises :

  • Python 3, Django ou FastAPI.
  • PostgreSQL, Git, Docker.
  • Esprit d'équipe et communication claire.

Note de JobsTracker :

Ce stage est idéal pour les étudiants de l'UL ou de l'IAEC souhaitant se spécialiser dans la Data Science.
